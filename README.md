# hustler-mob-api

This is the backend part (REST API) developed with PlayFramework of a system which will allow a person whatever their position to request any kind of service (Mechanics, Electricity, Plumbing,â€¦) if necessary. Agents who offer these services will also be able to make service offers.
Please see the documentation at https://www.playframework.com/documentation/latest/Home for more details.

## Running

Run this using [sbt](http://www.scala-sbt.org/).  If you downloaded this project from http://www.playframework.com/download then you'll find a prepackaged version of sbt in the project directory:

```
sbt run
```

And then go to http://localhost:9000 to see the running web application.
